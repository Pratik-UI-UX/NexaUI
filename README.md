# 🌌 Nexa UI
**The Aesthetic Engineering Ecosystem for Next.js 15.**

Nexa UI is a curated collection of premium, highly-optimized React components designed with a **macOS-first** philosophy. It bridges the gap between functional documentation and digital art.

---

## ✨ Features

* **🍎 macOS Aesthetic:** Native-feeling glassmorphism, "Dog" style docks, and Safari-inspired browser frames.
* **🧩 Bento-Grid Engine:** Sophisticated, asymmetric layouts with built-in parallax depth.
* **🔦 Spotlight Search:** A CMD+K interface that is blazing fast and visually stunning.
* **🎨 Tailwind CSS v4:** Leverages the latest CSS-variable-first engine for zero-runtime performance.
* **🎭 Motion-First:** Micro-interactions powered by Framer Motion (Morphing icons, spring physics).
* **🌓 Soothing Themes:** A custom "Cream" Light Mode (#FFFDF7) and "Deep Void" Dark Mode (#09090B).

## 📦 Installation

Nexa UI is designed to be easily integrated into any Next.js project.

```bash
# Install the core package
npm install @nexa-ui/core

# Or use the CLI to add specific components (Recommended)
npx nexa-ui add bento-grid
